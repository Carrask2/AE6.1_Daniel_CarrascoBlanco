 1. Descripción del Proyecto
Este proyecto consiste en el desarrollo integral de un sitio web para la marca de calzado "ALL SHOES". El objetivo principal ha sido crear una plataforma web funcional y visualmente coherente, aplicando conocimientos de maquetación, diseño responsive y estructura de navegación web.

 3. Estructura del Sitio (Navegación)
La web está compuesta por las siguientes secciones principales, accesibles desde el menú de navegación superior:

Inicio: Página principal que actúa como carta de presentación, integrando un sistema de slider (carrusel) para destacar novedades y tendencias de la marca.
Tienda: Catálogo interactivo de productos. Permite al usuario visualizar los modelos disponibles y gestionar la selección de artículos.
Nosotros: Sección corporativa donde se expone la identidad y misión de ALL SHOES.
Contacto: Espacio de comunicación que incluye un formulario de usuario para consultas personalizadas.

 4. Características Técnicas
Para el desarrollo de esta práctica se han aplicado los siguientes criterios:
Arquitectura: Diseño modular mediante el uso de archivos CSS separados para cada sección, garantizando un código limpio y escalable.
Adaptabilidad: Se ha implementado un diseño *responsive* (adaptativo) que asegura una correcta visualización en diferentes resoluciones de pantalla.
Interactividad: Uso de animaciones mediante CSS (efectos *hover*, transiciones y carruseles automáticos) para enriquecer la experiencia de usuario.
Identidad Visual: Consistencia en la paleta de colores corporativa y tipografía en todas las vistas de la aplicación.

 5. Instrucciones de Despliegue
El proyecto es compatible con cualquier servidor web estándar. Para su despliegue, se ha configurado el archivo `index.html` como punto de entrada raíz y se han organizado los activos estáticos (imágenes y estilos) mediante rutas relativas, garantizando la portabilidad del proyecto.
